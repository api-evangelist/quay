The initial release of the RISC-V Developer Preview, based on Red Hat Enterprise Linux (RHEL) 10.0, was in May 2025. Today, Red Hat is releasing a software refresh to that Developer Preview to update the code to RHEL 10.2. The hardware platform remains the same (SiFive HiFive Premier P550), but the new release contains more of the upstream code specifically for that platform as well as incremental upstream updates for support of the RISC-V instruction set architecture.
